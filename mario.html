Definiciones

Función
Conjunto de instrucciones que puedes definir una vez y reutilizar en diferentes partes de tu programa.
let
En Javascript es una palabra que se utiliza para declarar variables locales.
const
En Javascript es una palabra que se utiliza para declarar variables cuyo valor no cambiará una vez asignado.
<a>
Se utiliza para crear enlaces o hipervínculos en una página web. Permite a los usuarios hacer clic en el texto o la imagen enlazada para navegar a otra página web, archivo, ubicación en la misma página o realizar otras acciones específicas, como enviar correos electrónicos.
<footer>
Se utiliza para definir el pie de página de una página web. Generalmente se coloca al final del contenido de una página y contiene información como derechos de autor, enlaces a páginas relacionadas, datos de contacto o cualquier otra información que desees mostrar en la parte inferior de tu sitio web.
<section>
Se utiliza para dividir y estructurar el contenido de una página web en secciones o bloques temáticos. Cada sección puede contener contenido relacionado, como párrafos, encabezados, listas u otros elementos HTML. Esta etiqueta ayuda a organizar y etiquetar el contenido de una manera semántica, lo que facilita la comprensión y el estilo del diseño de la página.
